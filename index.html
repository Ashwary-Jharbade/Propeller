  • JavaScript library to rotate elements by mouse with inertia or without it.
  • Propeller uses requestAnimationFrame and GPU compositing for better performance. It is also compatible with touch devices.
  • Easy-to-use as jQuery plugin:
    $(nodeOrSelector).propeller(options);

    or zero-dependancy library
    new Propeller(nodeOrSelector, options)
  • Most valueble option is inertia. It is a number between 0 and 1. 0 means no rotation after mouse was released. 1 means infinite rotation. For this demo we use inertia equals to 0.99
